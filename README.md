# EduDash Pro - Mobile & Web Application

> Multi-tenant educational platform with advanced security, agentic AI features, and strict role-based access control (RBAC).

## Tech Stack

- **Mobile**: React Native (Expo)
- **Web**: Next.js 14 (App Router)
- **Backend**: Supabase (PostgreSQL + Auth + Edge Functions)
- **AI**: OpenAI GPT-4, Gemini, Claude
- **Payments**: PayFast (South Africa)
- **Video**: Daily.co

## Quick Start

```bash
# Install dependencies
npm install

# Start mobile app
npm start

# Start web app
cd web && npm run dev
```

## Documentation

For detailed documentation, architecture guides, and setup instructions, see:
**[edudash-pro-docs](https://github.com/EDP-APP/edudash-pro-docs)** (separate repository)

## Environment Setup

Copy `.env.example` to `.env` and configure:
- Supabase URL and keys
- OpenAI API key
- Daily.co API key
- PayFast credentials
